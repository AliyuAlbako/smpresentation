The Problem

Misinformation fuels crises, divides societies, and undermines peacebuilding in conflict-affected regions.

Challenges: linguistic diversity, fragile information systems, low digital literacy, and poor verification frameworks.

Our Solution — SmartCheckAI

✔️ AI-powered multilingual misinformation detection system

✔️ Classifies text as Fake or Real

✔️ Human-in-the-loop feedback verification

✔️ Provides credibility scores and insights

Key Features

✅ Multilingual AI Classification (BERT-based)

✅ Human Feedback System

✅ Actionable Outputs & Credibility Scores

✅ Current Accuracy: 85% on trimmed English dataset

✅ While hausa and other languages shows a signicance bias towards "fake news" class due to lack of much dataset to train model

How It Works

1️⃣ User submits news text or the news text through pipelines in real time

2️⃣ AI predicts credibility

3️⃣ User confirms or disagrees with feedback

4️⃣ Feedback stored for model improvement

Impact and Use Case

💡 Mitigates misinformation in fragile communities

💡 Supports diverse languages

💡 Empowers local validators

💡 Strengthens peacebuilding initiatives

Next Steps

🔸 Add more regional languages

🔸 Integrate audio & social media streams

🔸 Build mobile and web apps

🔸 Develop validator reputation system
